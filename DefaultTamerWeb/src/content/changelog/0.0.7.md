### A note on analytics

Default Tamer is a solo, free, open source project. Without any usage signal, it's genuinely hard to know what's working and what isn't — whether rules are being used, whether the chooser is getting in the way too often, or whether an update broke something for people. Crash reports don't tell the full story, and most users never file issues.

Opt-in analytics gives a way to answer those questions without compromising your privacy. If you choose to share, it makes a real difference in what gets prioritised next.

### Added

- Optional anonymous usage analytics you can enable in Preferences → Privacy. Helps improve the app — never collects your URLs, browsing history, or any personal information. Off by default.
- When a browser in a routing rule becomes unavailable, the app now detects and reports the failure anonymously (just a reason code — no browser names). This helps catch breakage faster.
- On startup, Default Tamer checks if the target browser for each rule is still installed. If a browser was uninstalled, the rule is automatically disabled and a warning notification appears. The Rules list also shows an orange warning badge on any rule with a missing browser, so you always know what needs attention.
- New Privacy Policy page at defaulttamer.app/privacy — explains exactly what is and isn't collected, event-by-event, in plain language.
- Privacy Policy link in the first-run setup and in Preferences, so you always know what you're opting into.
