### Added

- Initial release of Default Tamer
- Smart URL routing — send links to the right browser automatically based on the app they came from, the domain, or the full URL
- Hold `⌥ Option` while clicking any link to manually pick a browser on the fly
- Set a fallback browser for links that don't match any rule
- Manage and reorder rules with drag & drop
- Optional activity log — see exactly where every link was sent (URLs are sanitised before storage; off by default)
- Launch at login support
- First-run setup wizard to get you up and running in seconds
- Menu bar icon for quick access to your rules, recent routes, and preferences
